# Painel de Tarefas

Aplicação web de gestão de tarefas com CRUD completo.

## Tecnologias

- HTML
- CSS
- JavaScript
- Bootstrap
- Supabase (PostgreSQL)
- Git / GitHub
- Vercel

## Funcionalidades

- Criar tarefas
- Editar tarefas
- Concluir e reabrir tarefas
- Apagar tarefas
- Filtro por estado
- Data de criação automática
- Interface responsiva
